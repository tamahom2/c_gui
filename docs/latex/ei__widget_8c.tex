Creates a new instance of a widget of some particular class, as a descendant of an existing widget. 

The widget is not displayed on screen until it is managed by a geometry manager. When no more needed, the widget must be released by calling \mbox{\hyperlink{ei__widget_8c_aedbe2a89b8e68342362710a55ee1858b}{ei\+\_\+widget\+\_\+destroy}}.
